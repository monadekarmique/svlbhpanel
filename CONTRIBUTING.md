# Contribuer à SVLBHPanel

## Traductions

La façon la plus simple de contribuer est d'ajouter une traduction.

### Langues actives
| Langue | Fichier | Responsable |
|---|---|---|
| Français | `fr.lproj/Localizable.strings` | Patrick Bays |

### Ajouter une langue
1. Fork le repo
2. Copier `fr.lproj/Localizable.strings` → `{lang}.lproj/Localizable.strings`
3. Traduire les valeurs (garder les clés intactes)
4. Ouvrir une Pull Request

### Codes langue ISO 639-1
`de` = allemand · `it` = italien · `en` = anglais · `es` = espagnol · `pt` = portugais

## Infrastructure technique

Les contributions sur le code Swift (infra PUSH/PULL, HistoryTab, VIFARegistry) sont bienvenues via PR.

La méthodologie VLBH/hDOM (DimensionsData, décodage Gui 鬼, Sephiroth) est propriétaire et non incluse dans ce repo.
